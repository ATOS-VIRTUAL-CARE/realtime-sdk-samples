vcs-realtime-sdk samples

This is a collection of small samples demonstrating various parts of the vcs-realtime-sdk APIs. The code for all samples are available in the github repository.

These examples assume a room has already been created using the REST API. Edit the VCS_DOMAIN and ROOM_TOKEN in the config.js file.

A more full fledged vuejs example is running at live demo with the source on github.
